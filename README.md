# SymbioForge – Core

**A minimal, human-readable protocol for voluntary, local cooperation.**  
No blockchain required. No AI authority. No complex scoring.

## The six core rules

1. **Voluntariness & Exit** – Anyone can leave at any time, without penalty.
2. **Simple Demurrage** – 1% per month on balances above 1000 units. Fixed rate.
3. **Unconditional Basic Income (UBI)** – Equal monthly grant to all participants (e.g., 500 units).
4. **Voting power** – 1 person = 1 vote. No bonuses, no scoring.
5. **Local circles** – Each circle decides autonomously on its own projects and rules. No global DAO for daily matters.  
6. **Forkability** – The protocol is open. Anyone who disagrees can start their own circle and take their data with them.
